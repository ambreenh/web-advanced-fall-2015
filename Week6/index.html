JS Syntax

A computer program is a list of "instructions" to be "executed" by the computer. In a programming language, these program instructions are called statements. JavaScript is a programming language. JavaScript statements are separated by semicolons. JavaScript statements are composed of: Values (literals, values), Operators ( '='), Expressions (computation with values), Keywords (actions to be performed), and Comments.

JS Best Practices

// Declare at the beginning
var firstName, lastName, price, discount, fullPrice;

// Use later
firstName = "John";
lastName = "Doe";

price = 19.90;
discount = 0.10;

fullPrice = price * 100 / discount;
